Biller Application


Steps to build this project
1. Build the code using the following maven commands : mvn clean package -Dmaven.test.skip=true
2. Copy the generated 'billerApp-jar-with-dependencies.jar' from the target folder to the 'BillerApp' folder and rename it to 'billerApp.jar'

Once done , copy the 'BillerApp' folder to any Windows machine as a portable Biller App
Run the billerApplication.exe to run the file.
In case any exe license issue, directly execure the biller.bat filer

NOTE : the data is stored by default to the C:/BILLERDATA/ for this project
